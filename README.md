# To-Do Web App

This is a simple To-Do web application created using HTML, CSS and JavaScript.

## About the Project
This project helps users to manage their daily tasks. Users can add tasks, mark them as completed, and delete them easily.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Separate sections for Pending and Completed tasks

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Files
- todo_app.html → Main structure of the app
- todo.css → Styling of the app
- todo.js → Functionality and logic

## How to Run
1. Download all files
2. Open todo_app.html in any browser
